# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This repository contains course materials (lectures, workshops, assessments, resources) for **HCI 3900 — Human Computer Interaction** at ANU. Content is written in Markdown and built into HTML presentations (reveal.js) and PDFs (Beamer/LaTeX) using `pandoc`.

## Build Commands

Requires: `pandoc`, a TeX environment (MacTeX or TeX Live), `sass`, `python3`, and `make`.

| Command | Output |
|---|---|
| `make all` | Everything: reveal, beamer, assessments, resources, workshops, images, index |
| `make public` | Like `all` but excludes resources (used by CI/GitHub Actions) |
| `make html` | HTML only: reveal + assessments + workshops + resources |
| `make reveal` | Reveal.js HTML for lectures only |
| `make beamer` | Beamer PDF for lectures only |
| `make assessments` | Assessment HTML + PDF |
| `make workshops` | Workshop HTML only |
| `make resources` | Resource HTML only |
| `make bigfiles` | Concatenated mega-PDFs for all lectures, assessments, and workshops |
| `make clean` | Remove the `build/` output directory |

Build output goes to `build/`. The `build/index.html` is generated by `generate_index.py` from `_config.toml`.

CSS for reveal.js is compiled from SCSS:
```
sass css/charles_reveal_dark.scss build/lectures/charles_reveal_dark.css
```

## Content Structure

- `lectures/` — Slide decks (`.md`), built to reveal.js HTML + Beamer PDF
- `assessments/` — Assessment specs (`.md`), built to HTML + PDF
- `workshops/` — Tutorial/workshop activities (`.md`), built to HTML only
- `resources/` — Supplementary resources (`.md`), built to HTML only
- Each content directory has its own `img/` subdirectory — images **cannot be shared across directories**
- `references.bib` — Shared BibTeX references (APA citation style via `apa.csl`)
- `_config.toml` — Course metadata (title, institution, author, year) used by `generate_index.py`

## Markdown Conventions

Lecture slides use pandoc's Markdown with `--slide-level 2`:
- `#` headings create section title slides
- `##` headings create regular slides
- `###` and below are content within a slide

Reveal.js-specific syntax is supported (e.g., `{.columns}`, `{.column width="50%"}`, `background-image=` on headings).

Citations use pandoc format: `[@bibtex-key]`, `[@bibtex-key, p26]`. References go in `references.bib`; referencing style is APA (`apa.csl`).

## Python / Notebooks

The `notebooks/` directory contains Jupyter notebooks for data analysis demos used in lectures (e.g., survey analysis, statistical analysis). Dependencies are managed with Poetry:

```
poetry install
```

Dependencies: `pandas`, `seaborn`, `scipy`, `statsmodels`, `ipykernel`.

The `generate_index.py` and `find_unused_images.py` scripts use only the Python standard library.

## CI / Deployment

GitHub Actions (`.github/workflows/deploy.yml`) runs `make public` on push to `main` using the `pandoc/latex` Docker image and deploys the `build/` directory to GitHub Pages at <https://smcclab.github.io/thirty-nine-hundred-hci/>.
